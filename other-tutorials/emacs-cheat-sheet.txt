# Basics

Save:           C-x, C-s
Exit:           C-x, C-c
Undo:           C-x, u
Select text:    C-space
Cut:            C-w
Paste:          CMD + V
Cancel:         C-g
Close buffer:   C-x k
Other window:   C-x o      // useful for flipping to treemacs
Comment block:  M-;
Prev tab:       C-x b
Delete word:    M-DEL
Close window:   C-x 0
Find file:      C-c n   // Finds any file in the git repo

# IDE

Step into:      M-.
Go back:        M-,

# Treemacs
Open treemacs:  C-c t
Larger:         >
Smaller:        <
Delete:         d


# Customs

Reload config:  C-c r
Open config:    M-x cf   // TODO Make it a single command
Delete line:    C-c x
Copy text:      C-c c
indent:         TAB
Remove indent:  Shift + TAB
Comment line:   C-c ;
Rename:         C-c f6
Reformat:       C-c C-f
Next error:     C-c e

## Treemacs

Create Rust module in directory where cursor is:     C-c m
Create Rust package in directory where cursor is:    C-c p
Rename file:                                         C-f <f6>
Move file:                                           C-x <f6>  // Updates the mod.rs files also, but does not update imports yet

## Rust

reformat:       C-c C-f
create file:    C-c m   // Also updates the mod.rs
